# üõí Shopsphere

Shopsphere is a modern e-commerce web application built using React and Redux that enables users to browse products, manage cart items and wishlists, and view detailed product information with ease. The application is developed using technologies such as React, Redux, Framer Motion, Tailwind CSS, and Axios to ensure a smooth, responsive, and interactive user experience.

---

## üìå Features

- Browse products with smooth navigation  
- Add, remove, and clear cart items  
- Add and remove products from wishlist  
- Interactive carousel with pause and auto-slide functionality  
- Category-based filtering  
- Pagination (8 products per page)  
- Search across pages and categories with persistent results  
- Simulated login and signup using Fake Store API  
- Detailed product view page  
- Quantity management in cart
